Amigo Secreto

Este é um projeto simples de Amigo Secreto desenvolvido em JavaScript, HTML e CSS, permitindo a adição de amigos à lista, o sorteio de pares, a exclusão de amigos e a reinicialização do sorteio.

Funcionalidades

Adicionar amigos: Permite inserir os nomes dos amigos, evitando a duplicação e o preenchimento de campos vazios.

Sortear amigos: Após adicionar pelo menos 4 amigos, o sorteio é realizado de forma aleatória, exibindo os pares de amigo secreto.

Excluir amigos: É possível excluir um amigo da lista de amigos com um clique.

Reiniciar sorteio: Limpa a lista de amigos e o sorteio, permitindo recomeçar o processo.

Tecnologias Utilizadas

HTML: Estrutura da página.

CSS: Estilização da interface.

JavaScript: Lógica de interação, sorteio, validações e manipulação da lista de amigos.

Como Usar

Adicionar Amigos:

Insira o nome de um amigo no campo de entrada e clique no botão "Adicionar".

O nome será adicionado à lista de amigos.

Se o nome for repetido ou o campo estiver vazio, um alerta será exibido.

Sortear Amigos:

Após adicionar pelo menos 4 amigos, clique no botão "Sortear" para gerar os pares de amigo secreto.

Excluir Amigos:

Clique no nome de um amigo na lista para excluí-lo.

Reiniciar:

Clique no link "Reiniciar" para limpar a lista de amigos e o sorteio.

Como Funciona o Código
Funções

adicionar(): Adiciona um amigo à lista e garante que não haja nomes repetidos ou campos vazios.

sortear(): Embaralha a lista de amigos e exibe os pares no sorteio.

embaralhar(): Função de embaralhamento que reorganiza aleatoriamente os amigos na lista.

excluirAmigo(): Exclui o amigo da lista ao clicar sobre o nome.

atualizarLista(): Atualiza a exibição da lista de amigos na interface.

atualizarSorteio(): Limpa o resultado do sorteio.

reiniciar(): Reinicia a lista de amigos e o sorteio.
